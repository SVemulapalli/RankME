TASK: Evaluate the informativeness of the given utterance

The task is to evaluate the informativeness of the output of the automatic language generation system. 
Each utterance must be checked and rated.

Participants must be native speakers of British or American English.

Steps

In this experiment, you will evaluate the utterances that a tourist information system might say to someone looking for information about a place to eat. The overall goal of this experiment is to improve what the tourist information system says.

You will see the Meaning Representation and automatically generated utterance. The Meaning Representation is an abstract way of specifying the content which the system should convey.

In this task, you need to rate the utterance for its informativeness. 

Informativeness is defined as whether the utterance provides all the information given in the meaning representation and does not give any unnecessary information.

Examples

Example 1

This is a given Meaning Representation:
name[The Eagle], area[city centre]

This is a given utterance that needs to be evaluated:
The Eagle is a Chinese restaurant located in the city centre. 

Answer and explanation:

This utterance contains the information that is not provided in the meaning representation  ("Chinese restaurant"), so its informativeness score should not be very high.

Example 2

This is a given Meaning Representation:
name[The Eagle], area[city centre], pricerange[cheap]

This is a given utterance that needs to be evaluated:
The Eagle are located in the city centre. It is cheap.

Answer and explanation:

This utterance should be scored as highly informative - it contains all the information provided in the meaning representation.

Example 3

This is a given Meaning Representation:
name[The Eagle], area[city centre], near[Alimentum]

This is a given utterance that needs to be evaluated:
The Eagle is located near Alimentum.

Answer and explanation:

This utterance should be scored as of a low informativeness because it does not contain all the information provided in the meaning representation, the concept "area[city centre]" is missed.
