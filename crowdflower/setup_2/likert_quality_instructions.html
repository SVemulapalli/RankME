TASK: Evaluate the quality of the given utterance

The task is to evaluate the quality of the output of the automatic language generation system. 
Each utterance must be checked and rated.

Participants must be native speakers of British or American English.

Steps

In this experiment, you will evaluate the utterances that a tourist information system might say to someone looking for information about a place to eat. The overall goal of this experiment is to improve what the tourist information system says.

You will see the Meaning Representation and automatically generated utterance. The Meaning Representation is an abstract way of specifying the content which the system should convey.

In this task, you need to (1) tick the check-boxes that apply to a given utterance,  (2) rate the utterance for its quality. 

Quality is defined as an overall quality of the utterance, in terms of its grammatical correctness, fluency and other important factors.

Examples

Example 1

This is a given Meaning Representation:
name[The Eagle], area[city centre]

This is a given utterance that needs to be evaluated:
The Eagle is a Chinese restaurant located in the city centre. 

Answer and explanation:

The first check-box should be ticked because the utterance contains the information that is not provided in the meaning representation - "Chinese restaurant":

☑ The utterance contains additional information not present in the MR
☐ The utterance misses some information from the MR
☐ The utterance reflects the MR perfectly

Example 2

This is a given Meaning Representation:
name[The Eagle], area[city centre], pricerange[cheap]

This is a given utterance that needs to be evaluated:
The Eagle are located in the city centre. It is cheap.

Answer and explanation:

The last check-box should be ticked because the utterance contains all the information provided in the meaning representation and nothing additionally:

☐ The utterance contains additional information not present in the MR
☐ The utterance misses some information from the MR
☑ The utterance reflects the MR perfectly

This utterance should be scored as of a lower quality (the sentence is not grammatically correct).

Example 3

This is a given Meaning Representation:
name[The Eagle], area[city centre], near[Alimentum]

This is a given utterance that needs to be evaluated:
The Eagle is located near Alimentum.

Answer and explanation:

The second check-box should be ticked because the utterance does not contain all the information provided in the meaning representation, the concept "area[city centre]" is missed:

☐ The utterance contains additional information not present in the MR
☑ The utterance misses some information from the MR
☐ The utterance reflects the MR perfectly

Example 4

This is a given Meaning Representation:
name[The Eagle], area[city centre]

This is a given utterance that needs to be evaluated:
The Eagle is a Chinese restaurant. 

Answer and explanation:

The first and second check-boxes should be ticked because the utterance does not contain all the information provided in the meaning representation (the concept "area[city centre]" is missed) and at the same time it contains the information that is not provided in the meaning representation ("Chinese restaurant"):

☑ The utterance contains additional information not present in the MR
☑ The utterance misses some information from the MR
☐ The utterance reflects the MR perfectly
